# Сегментация зданий на ортофотопланах (U-Net)

## Описание проекта
В данном проекте реализована модель компьютерного зрения для автоматического выделения зданий на аэрофотоснимках (ортофотопланах).  
Задача решается с помощью семантической сегментации — для каждого пикселя изображения определяется, относится ли он к зданию или нет.
В дальнейшем модель доработается до выделения объектов МТ (Магистрального трубопровода).

Проект выполнен с использованием нейронной сети архитектуры **U-Net** и библиотеки **PyTorch**.

## Цель проекта
Разработать модель, способную:
- выделять здания на спутниковых/аэрофотоснимках
- строить бинарные маски зданий
- обобщаться на новые изображения

## Датасет
Использован датасет:
- Massachusetts Buildings Dataset (Kaggle)

Содержит:
- изображения (формат `.tiff`)
- соответствующие маски зданий (формат `.png`)

## Используемые технологии

- Python 3.9  
- PyTorch  
- Albumentations  
- NumPy  
- OpenCV  
- Matplotlib  
- Jupyter Notebook  

## Этапы реализации

### 1. Подготовка окружения
- создание виртуального окружения (conda)
- установка зависимостей
- запуск Jupyter Notebook в WSL

### 2. Подготовка данных
- загрузка изображений и масок
- организация структуры проекта
- разбиение на train / validation / test

### 3. Предобработка данных
- приведение изображений к RGB
- нормализация значений (0–1)
- уменьшение размера изображений до 256×256 / 512×512
- применение аугментаций:
  - горизонтальные отражения
  - вертикальные отражения

### 4. Создание Dataset и DataLoader
- реализация кастомного класса `Dataset`
- подготовка батчей для обучения

### 5. Построение модели
- реализация архитектуры **U-Net**
- encoder-decoder структура
- skip connections

### 6. Обучение модели
- функция потерь: Binary Cross Entropy (BCE)
- оптимизатор: Adam
- обучение на GPU (CUDA)

### 7. Валидация
- расчет loss на валидационном наборе
- отслеживание качества модели

### 8. Сохранение модели
- сохранение весов модели (`.pth`)
- возможность повторного использования без переобучения

### 9. Инференс
- загрузка обученной модели
- обработка новых изображений (PNG / JPG / TIFF)
- получение:
  - вероятностной маски
  - бинарной маски зданий

### 10. Визуализация результатов
- отображение:
  - исходного изображения
  - истинной маски
  - предсказанной маски

## Результаты

Модель успешно обучилась выделять здания на изображениях:

- наблюдается снижение функции потерь на обучении и валидации
- предсказания корректно отражают форму зданий
- модель обобщается на новые изображения
